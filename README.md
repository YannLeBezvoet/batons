# Batons
It's a project with stickmen and stickwomen who make stick things.
It's a side project; I'll never finish it.

Stick

## Controls
### Move Cursor
#### For azerty option
`q`: move left  
`d`: move right  
`z`: move up  
`s`: move down  
#### For qwerty option
`a`: move left  
`d`: move right  
`w`: move up  
`s`: move down  
### Move Camera
Use arrow keys
### Add stickmen
`0`: to add stickmen
### Add/Remove a block
`+`: to add block  
`-`: to remove block
